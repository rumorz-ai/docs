---
title: 'Get Entity Posts'
openapi: 'POST /get_entity_posts'
---

## Description

Retrieve posts related to specific entities within a given lookback period.

## Request

**POST** `/get_entity_posts`

### Headers

| Key         | Value          |
|-------------|----------------|
| Content-Type| application/json|
| X-API-Key   | YOUR_API_KEY   |

### Body Parameters

| Name      | Type     | Description                                      |
|-----------|----------|--------------------------------------------------|
| node_ids  | array    | List of node IDs to retrieve posts for           |
| lookback  | string   | Time period to look back (e.g., "1D", "3M")      |
| page      | integer  | Page number for pagination                       |
| limit     | integer  | Number of posts to retrieve per page             |

#### Example

```json
{
  "node_ids": ["7d8d81b3-0808-47ce-b459-a9fd5f74fd57"],
  "lookback": "1D",
  "page": 1,
  "limit": 10
}
```

## Response

### 200 OK

Successful response containing the posts related to the specified entities.

#### Example

```json
{
  "posts": [
    {
      "id": "post1",
      "content": "Sample post content",
      "timestamp": "2023-10-01T12:00:00Z"
    },
    {
      "id": "post2",
      "content": "Another sample post content",
      "timestamp": "2023-10-01T13:00:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total_pages": 1,
    "total_posts": 2
  }
}
```
