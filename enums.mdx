# Enums Documentation

## Lookback

Enum representing different lookback periods for retrieving data.

```python
class Lookback(Enum):
    ONE_HOUR = "1H"
    SIX_HOURS = "6H"
    TWELVE_HOURS = "12H"
    ONE_DAY = "1D"
    ONE_WEEK = "7D"
    ONE_MONTH = "30D"
    THREE_MONTHS = "90D"
    ONE_YEAR = "365D"
```

## SingleMetricTimeSeries

Model representing a single metric time series.

```python
class SingleMetricTimeSeries(BaseModel):
    metric: Union[EntityMetrics, FinancialAssetMetrics, str]
    values: List[Tuple[str, float]]
```

## TimeSeriesResponse

Model representing a time series response.

```python
class TimeSeriesResponse(BaseModel):
    entity_id: str
    time_series: List[SingleMetricTimeSeries]
```
